# Formularfragment erstellen

Ein Fragment ist ein wiederverwendbarer Teil eines Formulars. Beispielsweise kann ein Fragment einen Adressblock oder Copyright-Informationen enthalten. Die Verwendung von Fragmenten vereinfacht und beschleunigt die Erstellung und Pflege großer Formularbestände.


>[!VIDEO](https://video.tv.adobe.com/v/37325/quality=9)



Weitere Informationen finden Sie in der Dokumentation [zu](https://docs.adobe.com/content/help/en/experience-manager-65/forms/adaptive-forms-basic-authoring/adaptive-form-fragments.html) adaptiven Formularvorlagen

Bei Bedarf können Sie das Fragment &quot; [Spouse-details- ](assets/spouse-details-fragment.zip) Fragment&quot;herunterladen [und mithilfe desPaketmanagers in AEM importieren.](http://localhost:4502/crx/packmgr/index.jsp)
