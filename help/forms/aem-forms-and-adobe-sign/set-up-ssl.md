# SSL für AEM aktivieren

Eine der Voraussetzungen für die Integration von AEM Forms in Acrobat Sign besteht darin, Ihre AEM-Instanz für die Verwendung von SSL zu konfigurieren. Adobe Experience Managers SSL-Setup-Assistent erleichtert die Einrichtung einer AEM-Instanz, die über HTTPS ausgeführt werden kann.
Im folgenden Video werden die Schritte erläutert, die zum Konfigurieren von SSL mithilfe des Assistenten erforderlich sind
