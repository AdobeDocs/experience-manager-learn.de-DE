# Einrichten des Content-Hubs von AEM Assets

Erfahren Sie, wie Sie den Content-Hub von Experience Manager Assets mithilfe des Adobe Cloud Manager-Programms und des Assets-Add-ons einrichten. Erfahren Sie auch, wie Sie den Zugriff für Benutzende mithilfe von Produktprofilen konfigurieren.

>[!VIDEO](https://video.tv.adobe.com/v/3472918/?learn=on)


## Einrichten von OpenAPIs für AEM Assets Dynamic Media

Wenn Sie über eine Lizenz für AEM Assets Dynamic Media OpenAPIs verfügen, können Sie diese zum Generieren dynamischer Ausgabedarstellungen in Content Hub verwenden, z. B. für smartes Zuschneiden. Bevor diese Dynamic Media OpenAPI-Ausgabedarstellungen in Content Hub angezeigt werden können, ist eine Ersteinrichtung erforderlich.
