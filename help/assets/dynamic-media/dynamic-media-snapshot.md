# Erfahren Sie mehr über die Vorteile von Dynamic Media für Bilder {#snapshot}

Dynamic Media Snapshot ist ein visuelles Demonstrationswerkzeug, das die Leistungsfähigkeit von Dynamic Media für eine optimierte und dynamische Bildbereitstellung veranschaulicht. Experimentieren Sie mit Testbildern oder Dynamic Media-URLs, um die Ausgabe verschiedener Dynamic Media-Bildmodifikatoren visuell zu beobachten, und optimieren Sie die intelligente Bildbearbeitung für die Dateigröße (mit WebP- und AVIF-Bereitstellung) sowie das Netzwerk- und Geräte-Pixel-Verhältnis (DPR).
