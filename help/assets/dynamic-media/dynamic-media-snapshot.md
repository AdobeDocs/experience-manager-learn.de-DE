# Adobe-Momentaufnahme {#snapshot}

Adobe Snapshot ist ein visuelles Demonstrations-Tool, das die Leistungsfähigkeit von Dynamic Media für eine optimierte und dynamische Bildbereitstellung veranschaulicht. Experimentieren Sie mit Testbildern oder Dynamic Media-URLs, um die Ausgabe verschiedener Dynamic Media-Bildmodifikatoren, intelligenter Zuschnitte und intelligenter Bildoptimierung für die Dateigröße visuell zu beobachten (mit WebP- und AVIF-Bereitstellung).
