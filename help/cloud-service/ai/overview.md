# KI in Experience Manager

![KI in Experience Manager](./assets/overview/hero.png)

Experience Manager as a Cloud Service bietet erweiterte KI-Funktionen zur Verbesserung des Content-Managements, zur Optimierung von Workflows und zur Verbesserung von Benutzererlebnissen. Durch die Integration von KI-gestützten Funktionen wie dem AEM-KI-Assistenten und den AEM-KI-Agenten können Benutzende Aufgaben automatisieren, Erkenntnisse gewinnen und die Inhaltsbereitstellung optimieren.
